## Ecossistema de Produtos

Todos os projetos abaixo foram desenvolvidos para atender uma **operação de telemarketing com mais de 90 operadores**. Cada ferramenta desempenha um papel específico no fluxo de atendimento, trabalhando em conjunto para garantir eficiência, conformidade e qualidade no serviço prestado.

| Projeto | Descrição | Tecnologias | Link |
|---|---|---|---|
| **SupPaciente** | Sistema avançado de gestão de chamados em tempo real com triagem integrada, monitoramento de operadores e painéis administrativos. Arquitetura SPA com Firebase RTDB. | HTML5, CSS3, JS, Firebase | [Acessar](https://sup-paciente.vercel.app/) |
| **DocBox** | Ferramenta inteligente para registro e cópia padronizada de atendimentos. Extração automática de dados via área de transferência, validação de CPF/CNPJ e auto-save por aba. | HTML5, CSS3, JS Vanilla | [Acessar](https://doc-box.vercel.app/) |
| **DocBox Pro** | Versão profissional do DocBox com autenticação, painel administrativo de scripts, histórico de atendimentos e integração com Supabase em tempo real. | HTML5, CSS3, JS, Supabase, Vite | [Acessar](https://docbox-pro.vercel.app/) |
| **LGPD** | Ferramenta de conformidade para transferência segura de dados cadastrais. Extração inteligente de contatos com mascaramento de dados sensíveis. | HTML5, CSS3, JS | [Acessar](https://casemiro-dev.github.io/LGPD/) |
| **Painel de Scripts** | Dashboard interativo para consulta de scripts de atendimento em tempo real. Sistema de avisos, sugestões da equipe e atualização instantânea via Supabase Realtime. | HTML5, Tailwind CSS, JS, Supabase | [Acessar](https://casemiro-dev.github.io/Painel-de-Scripts/) |

### Como os projetos se conectam

O fluxo de atendimento na operação utiliza estas ferramentas de forma integrada:

1. **SupPaciente** gerencia a fila de chamados e distribui os atendimentos entre os operadores
2. **DocBox / DocBox Pro** é usado para registrar cada atendimento de forma rápida e padronizada
3. **Painel de Scripts** fornece os scripts corretos para cada tipo de solicitação
4. **LGPD** garante que a validação de contatos siga as diretrizes de privacidade
